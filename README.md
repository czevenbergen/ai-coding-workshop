# AI Coding Workshop — Stakeholder Directory

A workshop project for learning AI-assisted coding with GitHub Copilot.

## Quick Start

```bash
uv sync
uv run fastapi dev app/main.py
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

## What's in this folder

| File | What it is |
|---|---|
| `step-by-step-guide.md` | Your guide for the workshop — follow this during the day |
| `preparation-guide.md` | Setup instructions (you should have done this already) |
| `app/` | The starter application — includes sample stakeholder data and templates |
